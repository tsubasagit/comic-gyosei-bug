# 第001話「ようこそ特務トレンド係へ」

## あらすじ（3行）

1. 高校生バイトのショータが稲城市役所「特務トレンド係」に初出勤し、欲望まみれのテツ、無感情AIのミーム、苦労人リーダーのレナと出会う。
2. テツが流行中の「#全力土下座チャレンジ」を市の公式SNSでバズらせようと企み、ミームに最適化を指示したところ、ミームが暴走して市役所前で市民を巻き込んだ大規模土下座パフォーマンスを勝手に開催。
3. 動画がバズりすぎて「市役所が市民に土下座を強要」と大炎上。報告を受けた豪徳寺課長がマッスル覚醒し、テツに鉄拳制裁を下す。

## 登場キャラクター

- [x] 源田テツ（主人公・金の亡者）
- [x] ミーム（AI・暴走する）
- [x] 赤嶺レナ（リーダー・ツッコミ＆制止役）
- [x] 鈴木ショータ（語り手・読者視点・初出勤）
- [x] 豪徳寺剛（課長・ラスボス的登場）
- その他：市役所の人事担当（モブ）、市民（モブ）

## テーマ / 元ネタトレンド

「#全力土下座チャレンジ」— SNSで流行する○○チャレンジ系のバズ動画。土下座の角度・美しさ・勢いを競うネタチャレンジ。これを行政が公式採用しようとして暴走する。
