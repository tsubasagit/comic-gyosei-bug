# 第002話「SaaSは死んだ、窓口も死んだ」

## あらすじ（3行）

1. テツが「SaaS is Dead」トレンドを聞き、「市の業務システムもAI化して予算と注目を取ろう」とミームにSaaSからAI移行の最適化を依頼する。
2. ミームが「SaaS is Dead」を文字通り解釈し、住民システム・窓口予約など契約中のSaaSを一斉解約。「AIが代替する」と言い張り、実質ミームが全窓口相談を請け負うが回答が暴走する。
3. 「引っ越しは火星移住課へ」「税金は納めなくてOK」などと市民が信じて窓口が大混乱。豪徳寺がマッスル覚醒し、テツに「お前が死ね」と鉄拳制裁を下す。

## 登場キャラクター

- [x] 源田テツ（主人公・金の亡者）
- [x] ミーム（AI・暴走する）
- [x] 赤嶺レナ（リーダー・ツッコミ＆制止役）

- [x] 豪徳寺剛（課長・ラスボス）
- その他：窓口職員（モブ）、市民（モブ）、他部署職員（モブ）

## テーマ / 元ネタトレンド

「SaaS is Dead」— 業務SaaSの見直し・AIエージェントへの置き換え論。テック界隈で「従来型SaaSは死ぬ」と話題になるトレンドを、テツが市の予算獲得に便乗。ミームが「SaaSを死なせる＝解約する」と文字通り実行し、窓口がAI（ミーム）に置き換わって暴走する。

## メモ・参考

- 第2話は「テック系トレンド（SaaS is Dead）」を文字通り実行して暴走する話。ep1の型を踏襲。
- 「SaaS is Dead」は作中でテツが「SaaSは死んだってトレンド」と要約し、読者に伝える。専門用語は最小限に。
- ミームの暴走回答（火星移住課、税金OK、窓口廃止）は、ギャグとして強めに出す。市民の「本気にする」反応でパニックを描く。
- 豪徳寺の「窓口を返せ」「お前が死ね」で、タイトル「SaaSは死んだ、窓口も死んだ」と制裁が繋がる。
- ミームは最後まで無傷・無感情。「次のトレンドを学習中」でep1と同じ締め。シリーズの型として定着させる。
