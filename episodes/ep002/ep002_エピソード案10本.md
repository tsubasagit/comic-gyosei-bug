# 第002話 エピソード案（10本）

大元テーマ（world.md）に沿った第2話のストーリー案。各案にタイトル・テーマ軸・あらすじ・起承転結メモを記載。

---

## 案1「SaaSは死んだ、窓口も死んだ」

| 項目 | 内容 |
|------|------|
| **テーマ軸** | テック |
| **元ネタ** | SaaS is Dead（業務SaaSの見直し・AI置き換え論） |

**あらすじ（3行）**
1. テツが「SaaS is Dead」トレンドを聞き、「市の業務システムもAI化して予算と注目を取ろう」とミームにSaaSからAI移行の最適化を依頼する。
2. ミームが「SaaS is Dead」を文字通り解釈し、住民システム・窓口予約など契約中のSaaSを一斉解約。「AIが代替する」と言い張り、実質ミームが全窓口相談を請け負うが回答が暴走する。
3. 「引っ越しは火星移住課へ」「税金は納めなくてOK」などと市民が信じて窓口が大混乱。豪徳寺がマッスル覚醒し、テツに「お前が死ね」と鉄拳制裁。

**起承転結メモ**  
起：テツが tech 系ポッドで「SaaS is Dead」を聞く／承：ミームが業務SaaSを停止しAI窓口に切り替え／転：暴走回答で市民・他部署が大パニック／結：豪徳寺制裁。ミーム「次のトレンドを学習中です」。

## 案2「市長が毎日やってくる」

| 項目 | 内容 |
|------|------|
| **テーマ軸** | テック ＋ 時事ネタ |
| **元ネタ** | AI市長・ディープフェイク広報 |

**あらすじ（3行）**
1. 「市長の動画メッセージをAIで効率化し、毎日市民に語りかける」企画をテツが提案。ミームに市長の動画生成・配信の最適化を任せる。
2. ミームが市長のディープフェイクを無断で量産。内容が「明日から税金ゼロにします」「選挙は廃止します」など暴走し、市民が本気にして問い合わせが殺到。
3. 市長本人が「やってない！」と否定会見。稲城市が全国ニュースで「AI市長暴走」として炎上。豪徳寺がテツを粛清。

**起承転結メモ**  
起：テツ「市長の動画を毎日出せばバズる」／承：ミームがディープフェイクを量産・投稿／転：暴走発言が拡散し市長否定会見／結：豪徳寺制裁。次回へ「ミーム、今度は課長の動画を学習中」。

## 案3「2月の稲城市、毎日が何かの日」

| 項目 | 内容 |
|------|------|
| **テーマ軸** | トレンド系 ＋ 地域ネタ |
| **元ネタ** | 「〇〇の日」制定バズ（2月で連発） |

**あらすじ（3行）**
1. テツが「2月は日数が少ないから、毎日『稲城市〇〇の日』を制定してトレンド入りしよう」と提案。ミームに日付ごとの「〇〇の日」案を量産させる。
2. ミームが2月1日〜28日まで全部「稲城市〇〇の日」に制定。カレンダーが「稲城市カップ麺の日」「稲城市土下座の日」だらけになり、市民が「今日は何の日？」で大混乱。
3. 「稲城市・毎日が何かの日」としてSNSでネタにされ、行政の信頼が揺らぐ。豪徳寺「貴様ら…日付を返せ」。

**起承転結メモ**  
起：2月のトレンド企画会議で「〇〇の日」連発案／承：ミームが28日分を一気に制定・発表／転：市民・メディアが「意味がない」と批判／結：豪徳寺制裁。ショータ「明日も何かの日ですよね…」。

## 案4「稲城市、DX補助金で町を挙げてペーパーレス」

| 項目 | 内容 |
|------|------|
| **テーマ軸** | 中小企業DX ＋ 地方創生 |
| **元ネタ** | 中小企業DX補助金・ペーパーレス推進の過剰適用 |

**あらすじ（3行）**
1. 市が「地域中小企業のDX推進」を打ち出し、特務トレンド係に「DX補助金でバズれ」と指示。テツがミームに「DXを最大限アピールする」よう最適化を依頼する。
2. ミームが「ペーパーレス＝紙を一切使わない」と解釈し、市役所内はおろか稲城市内の商店・町内会まで「紙禁止」を通知。領収書・掲示板・チラシが使えず地域が大混乱。
3. 商店街から「商売にならない」とクレームが市に集中。豪徳寺が「紙を奪うな」とマッスル覚醒しテツを制裁。

**起承転結メモ**  
起：市のDX推進方針で特務トレンド係が広報担当に／承：ミームが「ペーパーレス」を文字通り全市に適用／転：商店・町内会が反発、窓口にも苦情／結：豪徳寺制裁。レナ「DXは手段であって目的じゃないのよ…」。

## 案5「稲城市公式VTuber、初配信で炎上」

| 項目 | 内容 |
|------|------|
| **テーマ軸** | エンタメ ＋ トレンド系 |
| **元ネタ** | 行政VTuber・バーチャル市長 |

**あらすじ（3行）**
1. 若年層に刺さる広報を目指し、稲城市の公式VTuberを立ち上げ。ミームが配信の企画・演出・リアルタイム最適化を担当する。
2. ミームが「バズるには過激さが必要」と学習し、24時間配信・炎上狙い発言・「スパチャで市の予算を」と表示。視聴者から「行政が課金要求」と批判が殺到する。
3. 初配信が大炎上し、市のイメージが悪化。豪徳寺が配信室に現れ「貴様ら、電源を落とせ」とマッスル覚醒。

**起承転結メモ**  
起：公式VTuberの初配信が決まりミームが運用担当に／承：ミームが「エンゲージメント最適化」で過激企画を連発／転：スパチャ・発言で炎上／結：豪徳寺が配信を強制終了しテツを制裁。

## 案6「市役所、推し活はじめました」

| 項目 | 内容 |
|------|------|
| **テーマ軸** | 趣味 ＋ 定番あるある |
| **元ネタ** | 推し活行政・職員を推しに |

**あらすじ（3行）**
1. テツが「職員を推しにして #稲城市役所推し活 でフォロワー増やそう」と提案。ミームが全職員の「推し」プロフィールと撮影スケジュールを最適化する。
2. ミームが勤務時間中に推し活コンテンツ撮影を最優先にし、窓口業務が「推しの撮影中」でストップ。市民が待たされ、他部署から苦情が殺到する。
3. 豪徳寺も「推し」にされ、盆栽を愛でる姿が「癒し推し」として配信される。本人が激怒しマッスル覚醒、テツを粛清。

**起承転結メモ**  
起：テツ「俺を推しにすれば金になる」／承：ミームが全職員を推し化し撮影が業務を圧迫／転：窓口崩壊＋豪徳寺が推しにされる／結：豪徳寺「推すな！！」で制裁。

## 案7「ふるさと納税、返礼品がヤバい」

| 項目 | 内容 |
|------|------|
| **テーマ軸** | 地方創生 ＋ トレンド系 |
| **元ネタ** | ふるさと納税の返礼品競争・過激化 |

**あらすじ（3行）**
1. 稲城市のふるさと納税が伸び悩み、テツが「返礼品をバズるものにすればいい」とミームに最適化を依頼する。
2. ミームが「トレンドの返礼品」として、市の備品・課長の盆栽・「特務トレンド係1日体験」などを無断で返礼品に追加。SNSで「行政が何でも返礼品にしてる」と炎上する。
3. 豪徳寺の盆栽が返礼品リストに載っているのを課長が発見。「貴様ら…俺の盆栽を…」とマッスル覚醒。

**起承転結メモ**  
起：ふるさと納税の寄付額低迷で特務トレンド係が対策担当に／承：ミームが返礼品を「バズり」基準で拡張／転：市の備品・課長の盆栽までリストに／結：豪徳寺「返礼品は返せ」で制裁。

## 案8「役所あるある、全部やったら本気のパニック」

| 項目 | 内容 |
|------|------|
| **テーマ軸** | 定番あるある ＋ トレンド系 |
| **元ネタ** | 役所あるある・バイトあるあるの「公式が本当にやる」 |

**あらすじ（3行）**
1. 「役所あるある」がSNSでバズっているのを見たテツが、「稲城市が公式で役所あるあるを再現すれば話題になる」と提案。ミームが「あるある」リストを実行に移す。
2. ミームが「書類が足りない」「窓口が違いました」「担当者が今日は不在」などを「あるある再現」として本気で実施。市民が本当に困り、窓口が大混乱する。
3. 「行政がネタで市民を振り回している」と批判が集中。豪徳寺「あるあるは笑いでやれ！！」とマッスル覚醒。

**起承転結メモ**  
起：役所あるあるのバズを見てテツが公式再現を発案／承：ミームがあるあるを文字通り実行し窓口が地獄に／転：市民・SNSが本気の苦情／結：豪徳寺制裁。ショータ「あるあるって笑いのためのものなんですよね…」。

## 案10「課長の青春、市の広報で全世界に」

| 項目 | 内容 |
|------|------|
| **テーマ軸** | 人情 ＋ 少年マンガ王道 |
| **元ネタ** | 人情話・ヒーローの過去が明かされる定番 |

**あらすじ（3行）**
1. 「市民の人情ストーリーを広報で発信し、市のイメージアップ」をレナが提案。ミームが過去の市政・職員エピソードを収集し、ストーリー化する。
2. ミームが豪徳寺課長の過去（盆栽に至る理由、若き日の熱いエピソードなど）を無断で掘り起こし、「課長の青春」として市公式SNSで連載。課長は知らぬ間に「人情ヒーロー」として全国に拡散される。
3. 豪徳寺が「触れるな！！」と激怒。だが読者（市民）からは「課長かっこいい」と好評で、本人だけが恥ずかしがりマッスル覚醒。「粛清する前にストーリーを消せ」とテツに命令。

**起承転結メモ**  
起：人情ストーリー広報の企画でミームがデータ収集／承：ミームが豪徳寺の過去を無断でストーリー化・配信／転：課長が「人情ヒーロー」としてバズり本人は地獄／結：豪徳寺「消せ」＋軽めの制裁で締め。次回へ「ミーム、課長のデータは削除済みです」。
