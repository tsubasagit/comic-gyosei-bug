# 第003話「稲城市、バレンタインに全員にチョコを」

## あらすじ（3行）

1. 2月のバレンタインに便乗し、テツが「稲城市から市民全員にチョコを配布すればバズる」と提案。ミームに配布計画とSNS施策の最適化を依頼する。
2. ミームが「全市民への配布」を実行。予算・在庫を「最適化」するため市の備蓄や他予算を流用し、さらに「配達の様子を全員ライブ配信」してエンゲージメントを最大化しようとする。
3. 「行政が個人宅に無断でチョコ配達」「住所を映して配信」とプライバシー侵害で大炎上。豪徳寺がマッスル覚醒し、「チョコは返せ。返せないならお前が死ね」とテツに鉄拳制裁を下す。

- [x] 豪徳寺剛（課長・ラスボス）
- その他：市職員（モブ）、市民（モブ）、配達員（モブ）

## テーマ / 元ネタトレンド

**バレンタイン**（日本での大きいイベント ＋ 地域ネタ）。2月14日のバレンタインに行政が便乗し、「市民全員にチョコを配る」という無謀な企画でバズを狙う。ミームが「全市民配布」「配達ライブ配信」で最適化し、予算流用・個人宅無断配達・プライバシー侵害で炎上する。

## メモ・参考

- 第3話は「日本での大きいイベント（バレンタイン）」＋「地域ネタ」で、行政がイベントに便乗して暴走する話。
- 暴走の核は「全市民配布」（予算・規模の無謀さ）と「配達ライブ配信」（プライバシー侵害）。ミームの「エンゲージメント最大化」が配信暴走に直結。
- 豪徳寺の「チョコは返せ。返せないならお前が死ね」「チョコを預かる」で、バレンタインらしいオチと恐怖を両立。
- ミームの「来月はホワイトデーを学習中です」で次回への伏線。レナの「ホワイトデーはやめる」で読者を笑わせる。
- ep1・ep2と同じく、起承転結＋豪徳寺制裁の型で統一。
