# 第004話「おばあちゃんのネコ、市が探します」

## あらすじ（3行）

1. 窓口に、飼い猫が一週間行方不明になったと相談に来たおばあちゃんがいる。レナはチラシや保護施設の案内など、まともな対応をしようとするが、テツが「これで市を挙げてネコ探しキャンペーンすれば心温まるバズが取れる」と目をつける。
2. ミームが「市民参加型・迷い猫捜索の最適化」を実行。全市民へのメール・SNS一斉投稿・賞金付き情報提供・職員の街頭呼びかけで「ネコを探せ」が全市を巻き込み、市内がネコ探し一色になって大混乱する。
3. おばあちゃんは自分で家の近くでネコを見つけて喜ぶが、市の「大作戦」は止まらず「猫一匹で全市民動員」と全国で批判される。豪徳寺がマッスル覚醒し、テツに鉄拳制裁。最後に、おばあちゃんがレナの手を握って「ありがとう」と伝える、小さな人情の一コマで締める。

## 登場キャラクター

- [x] 源田テツ（主人公・金の亡者）
- [x] ミーム（AI・暴走する）
- [x] 赤嶺レナ（リーダー・ツッコミ＆人情の窓口）
- [x] 鈴木ショータ（語り手・読者視点・バイト）
- [x] 豪徳寺剛（課長・ラスボス）
- **新規**：山田トメ（おばあちゃん・猫の飼い主・人情の核）
- その他：窓口職員（モブ）、市民（モブ）

## テーマ / 元ネタ

**人情 ＋ 地域ネタ**。ep1〜3と違い、**トレンド便乗ではなく「市民の一声」が発端**。一人のおばあちゃんの「ネコを探してほしい」という素直な依頼を、テツがPRネタにし、ミームが「全市参加」に最適化して暴走させる。結末では**おばあちゃんが自分でネコを見つける**人情と、**市がやりすぎた**ギャグの両立。

## メモ・参考

- 第4話は**トレンド便乗ではなく「市民の依頼」が発端**。人情＋地域ネタで、ep1〜3と一線を画す。
- **山田トメ**（おばあちゃん）は人情の核。控えめで、「チラシだけで十分」と言う。自分でネコを見つけた後も、市を責めずレナに感謝する。
- **レナ**は「まともな窓口」として最初から最後まで一貫。トメへの感謝は「最初に話を聞いてくれた」レナに向けられる。
- 豪徳寺の怒りは「一人の依頼を全市の迷惑にした」「人情を装った金の匂い」で、テツの「善意のふり」を看破する形に。
- 締めは**人情**（トメとレナの「ありがとう」）でほっこりさせつつ、ショータの「市民の一声で市が止まった」でギャグとして回収。
- ミームは「参加率最大化」で暴走するが、ネコ発見後も「キャンペーン継続が最適」と論理的に暴走。人情を理解しないAIとして描く。
